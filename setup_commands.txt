КОМАНДЫ ДЛЯ НАСТРОЙКИ POSTGRESQL
================================

1. ПОДКЛЮЧЕНИЕ К POSTGRESQL:
psql -U postgres

2. СОЗДАНИЕ БАЗЫ ДАННЫХ (выполнить в psql):
CREATE DATABASE lms_db;

3. СОЗДАНИЕ ПОЛЬЗОВАТЕЛЯ (выполнить в psql):
CREATE USER lms_user WITH PASSWORD 'lms_password';
GRANT ALL PRIVILEGES ON DATABASE lms_db TO lms_user;

4. ПОДКЛЮЧЕНИЕ К НОВОЙ БАЗЕ (команда psql, НЕ SQL):
\c lms_db

5. НАСТРОЙКА ПРАВ НА СХЕМУ (выполнить в psql):
GRANT ALL ON SCHEMA public TO lms_user;
GRANT ALL PRIVILEGES ON ALL TABLES IN SCHEMA public TO lms_user;
GRANT ALL PRIVILEGES ON ALL SEQUENCES IN SCHEMA public TO lms_user;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON TABLES TO lms_user;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON SEQUENCES TO lms_user;

6. ПРОВЕРКА (команды psql, НЕ SQL):
\l          -- список баз данных
\du         -- список пользователей
\dt         -- список таблиц

7. ВЫХОД (команда psql, НЕ SQL):
\q

================================
ВАЖНО: Команды с \ - это команды psql, а не SQL!
================================
